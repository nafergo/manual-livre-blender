Addons para modelação

Archimesh

Facilita a criação de interiores, adicionando alguns elementos arquitetónicos (i.e. portas, janelas, etc.).

Download aqui

ANT Landscape

Cria no menu Add>Mesh a opção Landscape que permite adicionar uma malha “paisagem”, um plano com aspecto de relevo (i.e. colinas, vales, etc.). Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros do objecto, modificando facilmente o aspecto do relevo, alterando o tipo de relevo.

Add Chain

Cria no menu Add>Mesh a opção Chain que permite adicionar um objecto constituído por uma “corrente” com dois aros. O objecto vem com uma curva que permite manipular facilmente a dimensão e disposição da corrente. Pode redimensionar a curva ou adicionar novos pontos por extrusão, em Edit Mode, modificando o número de aros, disposição da corrente, dimensão dos aros, etc. Se quiser converter a corrente para malha normal, seleccione a mesma e clique em ALT+C e escolha a opção Mesh From Curve. Depois pode apagar a curva, esta deixa de ser necessária, mas já não pode utilizar a curva para editar/manipular a corrente.

Bolt Factory

Cria no menu Add>Mesh a opção BOLT que permite adicionar parafusos e porcas. Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros dos objectos: modificar o aspecto, escolher tipos pré-definidos de porcas/parafusos, etc.

Extras

Cria no menu Add>Mesh o submenu Extras que permite adicionar novos objectos (i.e. estrela, etc.), diferentes dos primitivos originais (i.e. cubo, esfera, etc.). Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros relativos aos objectos adicionados (i.e. número de pontas da estrela, etc.).

Gears

Cria no menu Add>Mesh o submenu Gears que permite adicionar peças de engrenagem (i.e. roda dentada, etc.). Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros relativos aos objectos adicionados (i.e. número de dentes da roda dentada, etc.).

Gemstones

Cria no menu Add>Mesh o submenu Gemstones que permite adicionar objectos com forma de gema ou diamante. Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros relativos aos objectos adicionados.

Pipe Joints

Cria no menu Add>Mesh o submenu Pipe Joints que permite adicionar diversos objectos com formas de juntas de canos/canalização. Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros relativos aos objectos adicionados.

Regular Solids

Cria no menu Add>Mesh o submenu Solids que permite adicionar diversos objectos com a forma de sólidos geométricos menos comuns. Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros relativos aos objectos adicionados.

Twisted Torus

Cria no menu Add>Mesh a opção Twisted Torus que permite adicionar um objectos com a forma de um Torus (“donut”) torcido. A malha está disposta de forma diferente do Torus disponível no menu original. Na barra de ferramentas, surge um painel que permite manipular diversos parâmetros relativos ao objecto adicionado.
